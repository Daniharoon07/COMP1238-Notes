# Number row characters

- \% - Percent Sign 
- \^ - Caret 
- \& - And sign, Ampersand 
- \* - Asterisk, Star
- \( - Left Bracket, Left Parenthesis 
- \) - Right Bracket, Right Parenthesis 
- \_ - Underscore
- \+ - Plus sign
